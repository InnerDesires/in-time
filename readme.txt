Перед завантаженням вхідних даних необхідно: 
 - У кінець першого рядка файлу з даними заявок Printec додати "Плановий час МЦ|", без лапок.
 - Після цього формат першого рядка повинна виглядати наступним чином:
 Id заявки|Id банкомата|SN банкомата|Тип|Модель|Адреса|Місто|Область|Розташування|Локалізація|Опис заявки|Звіт|Стан заявки|Час створення|Час доступа|Плановий час|Фактичний час|Плановий час МЦ|
 - ПЗ звертається до даних за назвами вищеописаних полів, тому зміна формату призведе до некоректної роботи. 
 - Файли які завантажуються повинні мати кодування відмінне від win-1251|win-1252. Бажано використовувати UTF-8. Щоб змінити: 
	Бажаний файл відкриваємо у Notepad++ -> Кодування -> Перетворити у UTF-8 -> Тиснемо Ctrl + S

При підрахунку робочого часу використовується перебір по хвилинам, можлива похибка +-1 хв.
